Dicemon Tactic

Detail

           Dicemon Tactic is a multiplayer Monopoly-style strategy game where players roll dice, buy and own buildings, and use Dicemon skills to control the board.

Responsibility

           Developed core in-game systems, including the gameplay loop, minigame framework, and skill system. Implemented out-game systems such as asset bundle management and UI navigation flows for the Main Menu, Shop, Leaderboard, and other features.

Skill Graph

  The skill graph is implemented using XNode, a ScriptableObject-based NodeGraph system. I designed it to make Dicemon skills easier to create and maintain using Unity Addressables, while also making it easier to manage command-flow execution between the dedicated server and game client.
