Webcamfight

Detail

           Development of Webcam-Based Online Fighting Game (Webcam Fight) is an online fighting game using webcam (OpenCV) and machine learning (SVM) to process and predict the user's motion. Therefore, user can control character by webcam. The motion in the game have 6 motions such as Left Jab, Left Uppercut, Left Dodge, Right Jab, Right Uppercut and Right Dodge. There are 3 modes in this game: Tutorial, Single Player and Multiplayer mode. Therefore, user can create or join room to play with another people via internet. Development of Webcam-Based Online Fighting Game (Webcam Fight) is developed using Unity3D, OpenCV, SVM, Photon network and Blender3D.And I develop this game by myself as senior project.
